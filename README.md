# xmr-crossplatform

    非x86平台挖矿。
    强制使用CPU後端，将x86专用代码改写成可移植。

## origin

    https://github.com/fireice-uk/xmr-stak

## tested

    已测试平台raspberry pi 2b(arm)
    已测试平台raspberry pi 3b(aarch64)
    已测试平台xiaoya(android-4.4.4)
    已测试平台tenda-ac18(arm with kernel 2.6.x)
